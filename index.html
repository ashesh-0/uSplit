uSplit: image decomposition for fluorescence microscopy

Ashesh, Alexander Krull, Moises Di Sante, Francesco Silvio Pasqualini, Florian Jug
Human Technopole
ICCV 2023
Paper Code PaviaATN Dataset

Summary of the work

In this work, the task is to decompose input patch into two output patches. A meta-architecture is proposed which is integrated with Hierarchical VAE and UNet variants. The core idea is to use multiple successively lower resolution images alongside the primary input patch. The constraint is that these lower resolution images have the primary input patch in their center. The motivation is to provide additional surrounding context to the network in a GPU memory efficient way.

The task: Decompose the input patch (left) into two output patches (right).
